Målsetning

Sosial netverkssida som skal fungera som eit platform for einsame i Noreg til å ta kontakt å få moglegheten til å tilbringa høgtidar eller generell kvardag med nokon. Siden skal fungera som eit anonymt nettverksida der folk stillar seg enten som tilgjengeleg som kontaktpersonar eller etterlysar kontakt. Anonymitet er viktig for sikkerhet og tryggleik for begge partar og siden skal då vera eit platform for trygg og lett måte å nå ut til andre for å så, om ynskjeleg, ta kontakt utanfor internett/siden.

Arbeidsplan

Oppretta ein oversiktleg og lett sida for brukarar av ensomt.no, hovudformålet er før jol og dens høgtid og so få det ut i presse og blant folk som ein mogleg hjelp for menneske som opplev einsamhet og sliter med det. Layout og plan for siden reint teknisk er noko som utviklarane og dei relevante partane må planlega og jamføra med kvarandre.

Etiske hensyn

Sosiale nettverksider har sine problem med misbruk og unønskete element. Det er potensial og fare for at nokon kan bruke eit sånt plattform for uredelege og kriminelle hensikt som er derfor ein har anonymitet. Likevel må det tenkast på kva andre tiltak man kan gjera for å forhindre på best mogleg vis å luke ut sånne unønskede element.
